# DESIGN.md

## 项目与用户画像
- 美食探索类 Web 应用，整合大众点评与高德扫街榜数据
- 目标用户：寻找周边上榜美食的本地生活消费者
- 核心场景：移动端优先，户外行走中快速查找附近美食

## 品牌与视觉方向
- 意象锚点：黄昏时分的美食街——暖黄灯光映在石板路上，蒸汽从锅灶中升腾，空气中弥漫着食物的香气
- 气质关键词：烟火气、温暖、可信赖、探索感
- 避免：冰冷的科技蓝、过度饱和的荧光色、廉价促销感

## Design Tokens

### 色彩
- 主色：暖橙红 `#E85D3A`（灵感：炭火烤制的暖光）
- 辅助色：金黄 `#F5A623`（灵感：油炸食物的酥金色）
- 背景色：暖白 `#FAF7F2`（灵感：宣纸质感）
- 深色背景：`#1A1A2E`（用于地图区域，灵感：夜空）
- 文字主色：`#2D2D2D`
- 文字次色：`#6B7280`
- 成功/好评绿：`#10B981`
- 数据来源标识 - 大众点评红：`#FF6633`
- 数据来源标识 - 高德蓝：`#3B7FFF`

### 字体
- 标题：PingFang SC, Noto Sans SC, sans-serif
- 正文：PingFang SC, Noto Sans SC, sans-serif
- 数字：DIN Alternate, Tabular Nums

### 圆角
- 卡片：16px
- 按钮：12px
- 标签：8px
- 头像/图标：50%

### 阴影
- 卡片：`0 2px 12px rgba(0,0,0,0.06)`
- 悬浮卡片：`0 8px 30px rgba(0,0,0,0.12)`

## 布局与响应式
- 移动端优先设计
- 地图占据上半屏（约 45% 高度），列表在下半屏可滚动
- 桌面端：左侧地图 + 右侧列表面板（双栏布局）
- 筛选栏吸顶固定

## 交互与状态
- 卡片点击有轻微缩放反馈（scale 0.98）
- 列表项进入视口时渐入动画
- 距离筛选切换时列表有过渡效果
- 加载状态使用骨架屏

## 设计禁忌
- 不要使用纯蓝色调（太冷，不符合美食氛围）
- 不要使用大面积渐变背景
- 不要使用过于圆润的气泡风格
- 不要使用促销类红色大字报风格
- 避免信息密度过高，保持呼吸感
